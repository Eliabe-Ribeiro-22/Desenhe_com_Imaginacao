Jogo para desenhar usando toda a sua imaginação.

## Comandos do jogo:
Click do Mouse para pintar - left and right
- Teclado:
R = Limpar desenho
A = Tocar sinos
Alt + F4 = fechar o jogo
  - Alterar cor do Pincel Esquerdo
	s = Amarelo
	c = Green
	w = Rosa
	e = Roxo
	q = vermelho
  - Alterar cor do Pincel Direito
	d = verde escuro
	z = vermelho
	x = preto
	b = azul
